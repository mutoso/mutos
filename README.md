# mutos
[![License: MPL 2.0](https://img.shields.io/badge/License-MPL%202.0-brightgreen.svg)](https://opensource.org/licenses/MPL-2.0)

An attempt at a microkernel written in Rust

# Building
(Requires cargo-xbuild, bootimage, and hjson)
```
./build.sh
```

# Running
(Requires qemu)
```
bootimage run
```
